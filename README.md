# wll-sdk-generator

[Docs](https://white-label-loyalty.github.io/wll-sdk)

## Getting started

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev --help
```

## Generating an SDK

```bash
bun dev gen https://api.staging.rewards.wlloyalty.net/v1/openapi.json -k YOUR_API_KEY
```
