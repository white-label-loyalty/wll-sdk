# WLL API SDK

## Get Started

To get started with the WLL API SDK check out the [docs]("https://white-label-loyalty.github.io/wll-sdk/TypeScript/intro")

## Installation

### npm

```bash
npm install @wll-sdk/api
```

### yarn

```bash
yarn add @wll-sdk/api
```

### bun

```bash
bun add @wll-sdk/api
```
